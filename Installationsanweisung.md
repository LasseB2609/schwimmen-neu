Zum Starten unserer Schwimmen-Anwendung bitte folgende Schritte durchführen.

1. Im Terminal in das Verzeichnis "schwimmen-neu/schwimmen-platform/server" navigieren und einmal den Befehl "npm install" ausführen.
2. Dann im Terminal zurück in das Verzeichnis "schwimmen-neu/schwimmen-platform" springen und den Befehl "docker-compose up --build" ausführen, um die Container zu starten.
3. Die Website über localhost:8080 aufrufen und rechts im Registrieren Feld einen Account erstellen. Darauf wird zur Lobby-Seite weitergeleitet.
4. Auf der Lobbyseite kann eine eigene Lobby erstellt werden oder bestehen Lobbys beigetreten werden.
5. Der Lobby-Host kann das Spiel dann starten.

Weitere Informationen:
Auf die Datenbank kann über localhost:8085 zugegriffen werden. Username: "schwimmen_user"; Passwort: "schwimmen_pass"
